# Goal
Design landing pages that quickly establish value, trust, visual distinction, and a clear next action.

# Instructions
1. Clarify the audience, offer, conversion action, objections, and proof points.
2. Make the first viewport unmistakably about the product, place, person, or offer.
3. Build a clear narrative sequence: hero, proof, benefits, mechanics, social proof, objection handling, and conversion.
4. Use real or context-specific visuals whenever possible.
5. Keep CTAs specific and repeated at natural decision points.
6. Verify mobile composition, text hierarchy, image cropping, and scroll rhythm.

# Input
Use the product context, audience, brand tone, conversion goal, content assets, and technical stack.

# Output
Generate landing page layouts, components, copy structure, styling, assets, and responsive behavior.

# Best Practices
- Avoid generic hero-card layouts and vague value propositions.
- Make the headline literal enough to orient the user.
- Put supporting claims in concise copy backed by proof.
- Keep the next section visible or hinted when using full-viewport heroes.
- Optimize images, metadata, and performance for acquisition traffic.
